a) ¿Quién inventó el sistema de control de versión Git y por qué
Fue diseñado por  Linus Torvalds,  pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente

b) ¿A quién pertenece actualmente Github y por qué?
 Actualmente pertenece a Microsoft. El 4 de junio de 2018 Microsoft compró GitHub por la cantidad de 7500 millones de dólares1​2​, al inicio el cambio de propietario generó preocupaciones y la salida
 de algunos proyectos de este repositorio​, sin embargo no fueron representativos. GitHub continúa siendo la plataforma más importante de colaboración para proyectos Open Source.

Si desde la pagina web Github, o con entornos gráficos de git
