a. ¿Quién inventó el sistema de control de versión Git y por qué?
Linus Torvalds creo la primera revisión de git, un sistema de control de versiones (diseñado inicialmente para el desarrollo de Linux)
Con el proposito de tener la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas
tienen un gran número de archivos de código fuente.

b. ¿A quién pertenece actualmente Github y por qué?
Microsoft, porque la compañía junto con el cambio de CEO, adopto una estrategia nueva, mucho más abierta al mundo del software libre y
a la integración con otras plataformas y servicios.
