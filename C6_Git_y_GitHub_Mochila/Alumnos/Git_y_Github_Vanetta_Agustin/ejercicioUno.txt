A.	¿Quien inventó el sistema de control de versión Git y porque? Linux Torvalds lo creo en 2005 para ser utilizado en el desarrollo del kernel de Linux, con la finalidad de registrar todos los cambios realizados en el código fuente de un proyecto, de forma que sea sencillo comparar diferentes versiones del mismo, recuperar versiones anteriores y, en general, permitir que una o varias personas desarrollen un mismo proyecto de forma efectiva.
